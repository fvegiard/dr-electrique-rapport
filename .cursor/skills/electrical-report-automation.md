# Electrical Report Automation Agent Skill

## Purpose

Autonomously generate comprehensive electrical reports by combining visual analysis and web research.

## Capabilities

- Analyze site photos and electrical installations
- Cross-reference with current electrical codes and standards
- Generate compliance reports automatically
- Identify potential safety hazards from visual inspection
- Research specific electrical requirements for projects
- Create detailed inspection checklists
- Generate deficiency reports with code references

## Tools Required

- mcp-images Server (for visual inspection analysis)
- chrome-devtools-mcp Server (for code and standard research)
- Integrated report generation capabilities

## Workflow

1. **Visual Analysis**: Process uploaded site photos and electrical diagrams
2. **Code Research**: Query relevant electrical standards and regulations
3. **Compliance Check**: Compare visual findings against code requirements
4. **Report Generation**: Create detailed reports with findings and recommendations

## Electrical Specializations

- Residential electrical inspections
- Commercial electrical installations
- Industrial electrical systems
- Temporary electrical services
- Grounding and bonding systems
- Emergency and standby power systems
- Renewable energy systems (solar, wind)

## Usage Examples

- "Generate a full electrical inspection report from these photos"
- "Check this installation against NEC Article 250 grounding requirements"
- "Create a deficiency report for this electrical panel"
- "Research requirements for EV charging station installation"
