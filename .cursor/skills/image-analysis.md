# Image Analysis Agent Skill

## Purpose

Analyze images, screenshots, and visual content for electrical reports and documentation.

## Capabilities

- Extract text from images (OCR) for electrical diagrams and schematics
- Analyze electrical diagrams and circuit layouts
- Detect safety compliance issues in photos
- Generate reports from visual inspections
- Process uploaded images from electrical sites
- Identify electrical components and wiring patterns

## Tools Required

- mcp-images Server (for image fetching and processing)
- Vision analysis capabilities for electrical context

## Usage Examples

- "Analyze this electrical diagram for compliance"
- "Extract text from this safety inspection photo"
- "Identify components in this wiring schematic"
- "Check this site photo for electrical hazards"

## Electrical Domain Knowledge

- NFPA 70 (National Electrical Code) compliance checking
- Electrical symbol recognition
- Wiring diagram analysis
- Safety equipment identification
- Grounding system verification
- Load calculation validation
