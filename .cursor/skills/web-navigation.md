# Web Navigation Agent Skill

## Purpose

Navigate electrical codes, standards, and documentation websites autonomously for compliance and reference.

## Capabilities

- Search electrical standards (NFPA, NEC, CSA, IEC)
- Extract compliance requirements from official sources
- Monitor regulatory updates and code changes
- Generate compliance checklists from web sources
- Research electrical component specifications
- Access manufacturer documentation and datasheets
- Check for electrical permit requirements by jurisdiction

## Tools Required

- chrome-devtools-mcp Server (for web browsing and scraping)
- Web navigation and content extraction capabilities

## Usage Examples

- "Check the latest NEC requirements for grounding"
- "Find CSA standards for electrical installations in Quebec"
- "Research arc flash hazard calculations"
- "Get specifications for this electrical component"
- "Check local electrical code requirements"

## Electrical Domain Knowledge

- NFPA 70E (Electrical Safety in the Workplace)
- Canadian Electrical Code (CSA C22.1)
- International Electrotechnical Commission (IEC) standards
- Provincial/Territorial electrical regulations
- Electrical inspection requirements
- Equipment certification standards (UL, CSA, CE)
